## Requirements

  * PHP 8.1.0 or higher;
  * PostgreSQL module for PHP;
  * composer
  * and the [usual Symfony application requirements][1].

## Installation

```bash
$ cp .env-example .env
$ composer install
```

## Usage

```bash
$ symfony server:start
```
